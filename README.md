# KIMPVIEW

KIMPVIEW는 **김치프리미엄(김프)**과 주요 암호화폐 가격 정보를  
한 화면에서 직관적으로 비교할 수 있도록 만든 **개인 포트폴리오 프로젝트**입니다.

여러 거래소의 가격 데이터를 사용자 친화적인 UI로 정리하는 것을 목표로 했습니다.

---

## 📌 프로젝트 목적

- 거래소별 가격 차이를 한눈에 확인할 수 있는 화면 구성
- 프론트엔드 구조 설계 및 데이터 가공 로직 경험
- 단순 UI 구현이 아닌, **실사용을 고려한 화면 흐름 설계**

---

## 🛠 사용 기술

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- Upbit Open API
- Binance Open API
- WebSocket (실시간 거래 / 청산 알림)

---

## ✨ 주요 기능

- 주요 코인 가격 테이블 표시
- 업비트 / 바이낸스 가격 비교
- 김치프리미엄 자동 계산
- 실시간 거래 알림
- 실시간 강제 청산 알림
- 사용자 시선 흐름을 고려한 대시보드 레이아웃

---

## 🧩 구조 설계 포인트

- 데이터 영역과 UI 영역 분리
- 중복 API 호출 방지를 위한 캐시 구조 적용
- 실시간 알림 영역은 테이블 누적 방식으로 구현
- 실제 데이터 연동 전에도 화면이 깨지지 않도록 기본 UI 우선 설계
